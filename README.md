
# Solaar_AppImage

Solaar is a Linux manager for many Logitech keyboards, mice, and other devices that connect wirelessly to a Unifying, Bolt, Lightspeed or Nano receiver as well as many Logitech devices that connect via a USB cable or Bluetooth

https://github.com/pwr-Solaar/Solaar

## Repository: https://github.com/ryuuzaki42/Solaar_AppImage

## Notice
Need to be run as root to change Logitech Unifying 

### Run it as root
    su
    ./Solaar-*_JB-x86_64.AppImage

### Or
    sudo ./Solaar-*_JB-x86_64.AppImage

### If use KDE
    kdesu ./Solaar-*_JB-x86_64.AppImage
